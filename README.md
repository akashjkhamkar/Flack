# Flack---simple-chat-app
flack | a chat app made for teams using socketio and flask

## installation

install virtualenv

### 1 create environment 
    
    virtualenv [name]
    
### 2 activate
    
     source [name]/bin/activate
     
### 3 install dependencies

     pip install -r requirements.txt
     
## Run 

     flask run
